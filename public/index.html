Joe helped on the Front End with basic design layout using Materialize, HTML, and CSS. Joe's main role was to set up the AJAX requests sent to the Spoonacular API.

Weather Trek

Weather Trek is a full stack app which lets users input a location and date for a future trip. Once the user hits submit past weather history of that location and time are displayed to the user. Joe's main role in this project was setting up user authentication. Joe did this by learning and using Passport with Express and Sequelize.
