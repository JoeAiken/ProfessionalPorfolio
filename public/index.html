Get to know Joseph

Joseph Aiken is a Full-Stack Developer living in Charlotte, NC, seeking creative opportunities in Front End Development.
A graduate of UNC Charlotte's Full Stack program, and currently employed as a Front End Developer, Joseph has gained proficient skills in HTML, CSS, JavaScript, jQuery, Node.js, Express.js, MySQL and Git.
In his spare time Joseph enjoys playing guitar, hiking with his girlfriend and their dog, Piper, and attending shows in the lively Charlotte music scene.

Jam-Clash

Jam-Clash is a user driven, networking application built for musicians. A user can set up a profile account letting others know the instrument they play, if they're in a band, what musicians they are looking to jam with, and links to soundcloud, reverb profiles. Joseph worked on the front end team of the project. Materialize was used as the framework to give the application a dynamic, modern user experience.

Fridge 2 Plate

Fridge 2 Plate is an app which lets a user input ingredients they have in their fridge and provides recipes with videos that call for those ingredients. Joseph helped on the Front End with basic design layout using Materialize, HTML, and CSS. Joseph's main role was to set up the AJAX requests sent to the Spoonacular API.

Weather Trek

Weather Trek is a full stack app which lets users input a location and date for a future trip. Once the user hits submit past weather history of that location and time are displayed to the user. Joseph's main role in this project was setting up user authentication. Joseph did this by learning and using Passport with Express and Sequelize.
